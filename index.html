Repositories

  • HouseWeb

    This is a site where i practice some web technologies. It has the functions of logging in, exchanging messages, setting, editing and deleting notes and image gallery with a lightbox.

    The frontend is made in vue.js, the backend is an express application. Socket.io is used to send and receive messages.

    The site is deployed as a free web service on render.com, so after 15 minutes of inactivity it takes up to one minute for spinning up a service.

    For testing purposes, you can log in with "user" for both, username and password.

    Github repositories:

    • backend: https://github.com/matozu/houseWebBackend
    • frontend: https://github.com/matozu/houseWeb

    Deployed at: https://housewebbackend.onrender.com/
